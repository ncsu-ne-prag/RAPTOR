# XFTA

XFTA is a calculation engine for fault trees and related models written at the Open-PSA (XML) format
or using the domain specific language S2ML+PBE.

## Version

Current version is 2.0.1

## Installation

See INSTALL.txt

## Usage

xftar.exe script.xfta
